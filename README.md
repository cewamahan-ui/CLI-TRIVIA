# Trivia CLI Game

A command-line interface trivia game built with Node.js that tests your knowledge across timed questions.

## Features

- **Question Categories**: Kenyan History
- **Score Tracking**: Real-time score calculation
- **User Feedback**: Immediate feedback on answers
- **Game Statistics**: Performance analysis at game end
- **User-friendly Interface**: Clean, intuitive CLI interface

## Requirements

- Node.js (v12 or higher)

## Installation

1. Clone or download the repository
2. Navigate to the project directory
3. Run the game:

```bash
node trivia.js
